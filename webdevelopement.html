Pineapple Island

This was a cumulative assignment I completed with the Indigenous Friends Association. Students were asked to recreate a website based on an image and style guide. I was able to surpass the standard requirements by including a functioning form to submit emails for the website

Building the HTML file allowed me to showcase my knowledge and understanding of the skills we learnt over a twelve week period.

Creating the CSS file was challenging because I wanted to ensure that everything I made was scaled accurately in relation to the original image. This required a keen eye for detail and understanding of the styling semantics required to achieve the same look.
